The quotes project uses only the backend side of django-comments-xtd. Users can send comments, can reply to comments, and can send the like and dislike reactions to comments.

Threaded comments

Comments can be nested. The maximum thread level is customizable for the whole site and explicitly for each app.model pair.

Notifications and mute-links

Comments' authors can receive notifications of follow-up comments. Notifications can be cancelled by clicking on the mute-link.

Comment confirmation

Comment confirmation via email when comments' authors are not authenticated. Comments hit the database once they are confirmed.

Comment reactions

Starting with like and dislike reactions, they can be extended to any set of reactions to suit the comments of your website.
