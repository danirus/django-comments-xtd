This is the Django project that runs frontend tests. It is available in the directory js/tests/. It is made of an app called specs, that contains two models: LoggedOutSpec and LoggedInSpec.

Both models have a field called filename that refers to the test spec that runs when visiting each LoggedOutSpec or LoggedInSpec instance URL. The first model requires no authenticated user. The second model requires a logged in user, specified with the field user.

To run it manually, take a look at how the Gruntfile sets up the project.
